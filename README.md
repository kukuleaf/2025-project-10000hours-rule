1. 프로젝트 개요
  프로젝트 명: 1만 시간의 법칙 사이트
  설명: HTML, CSS 강의를 통해 배운 내용을 정리하고 실습한 예제 사이트입니다.
  목적: 학습한 웹 기술을 종합적으로 적용하여 웹사이트 구현을 경험하기 위함.

2. 주요 기능
  1) 다양한 시맨틱 태그 활용 (header, main, section, footer 등)
  2) JavaScript 기능
     - 버튼 클릭 이벤트 처리
     - 입력값 유효성 검사 (숫자 검사 포함)
     - 계산 기능 구현
     - 모달창 열기 / 닫기
  3) 반응형 웹 디자인
     - 1200px, 1024px, 768px, 480px, 360px 해상도 대응
  4) 폰트 적용
     - OTEnjoystoriesBA
     - GmarketSans
     - Noto Sans KR
  5) 기타 HTML, CSS 활용
